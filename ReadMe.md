## 一元配置分散分析表実験について
一元配置分散分析表実験は、3,4水準で作成しております。
### 一元配置分散分析実験の解析手順
1. データを収集する
2. データの構造式を考える
3. データをグラフ化する
4. 仮説を設定する
5. 平方和を計算する
6. 平方和の自由度を計算する
7. 分散分析表を作成する
8. 有意となった要因の効果を推定する
9. 最適水準における母平均を推定する
10. 最適水準における将来のデータを予測する
